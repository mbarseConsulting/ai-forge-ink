## Checklist

- [ ] Symmetry declaration matches the content — if "mutual" is declared, both perspectives should have comparable weight
- [ ] Power dynamic has a shift trigger — static power is a red flag for flat dynamics
- [ ] "What keeps A bound" names a specific mechanism, not a vague emotion
- [ ] Conflict triggers reference A's core Psychology (wound/vulnerability)
- [ ] "What this relation does to A" — growth/regression assessment is honest, not aspirational
- [ ] For toxic dynamics: the pathological mechanism is named, not euphemized
- [ ] For asymmetric relations: the gap in investment is explicit
- [ ] For posthumous/absent relations: what A still needs from B is stated
- [ ] Writing rules specify minimum 3 "never do" items specific to this pair
- [ ] Mirror file for B exists or is flagged as needed
