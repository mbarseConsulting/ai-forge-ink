## Loading

**Current snapshot:** {number} — {label} (or "baseline — no snapshots yet")

### Default loading — writing/ first

If `snapshots/{current}/writing/` exists (generated by `--compile`), load writing/ files instead of raw encyclopedia:
- `writing/voice.md` — how to write dialogue
- `writing/emotions.md` — how to write emotional expression
- `writing/relations.md` — how to write interactions with specific characters
- `writing/body.md` — when/how to describe physicality

These are actionable writing instructions derived from the encyclopedia. They replace the need to load `core/core.md`, `emotional-profile.md`, and `relations/*.md` for most scenes.

**Permanent guardrails:** Always also load `core/writing-rules.md` (always/never/traps/signals/deformation).

### When writing/ does not exist

Fall back to raw encyclopedia loading:
- `core/core.md` sections as needed
- `core/writing-rules.md`
- `snapshots/{current}/emotional-profile.md` (if exists)
- `snapshots/{current}/relations/*.md` (if relevant)

### Deep loading (pivotal scenes)

When a scene demands full psychological depth, load writing/ AND the relevant encyclopedia sections. writing/ provides the rules; the encyclopedia provides the why.

### Specific moment
Consult `index.md`, then read `core/` + `snapshots/{number}/`.
